Chủ đề: Showroom Ô tô (Car Dealership): Giao diện tối/sang trọng, tính năng so sánh các phiên bản xe.

Xây dựng website showroom ô tô frontend-only phong cách cinematic, sử dụng embed model 3D từ Sketchfab làm trung tâm (iframe + autostart, ẩn UI), kết hợp dark luxury UI, overlay text và animation (GSAP, scroll-based) để tạo trải nghiệm giống trailer quảng cáo xe, tập trung vào storytelling và hiệu ứng thị giác, không sử dụng backend.

Website showroom ô tô frontend-only theo phong cách cinematic: sử dụng model xe 3D thật từ Sketchfab bằng cách embed iframe trực tiếp (không cần xử lý 3D), cấu hình URL với các tham số như autostart=1, ui_controls=0, ui_infos=0 để tạo trải nghiệm tự chạy, tối giản UI và mang cảm giác cao cấp; phần trọng tâm không nằm ở model mà ở cách xây dựng giao diện xung quanh gồm dark luxury UI, overlay text, hiệu ứng scroll, animation (GSAP) và bố cục kể chuyện như trailer quảng cáo xe; toàn bộ hệ thống chỉ cần HTML, CSS, JS, không cần backend nhưng vẫn tạo được trải nghiệm thị giác mạnh, giống website hãng xe thực tế.

Yêu cầu chung (áp dụng cho các Project)
- Structure: Tổ chức thư mục rõ ràng (/css, /js, /images).
- HTML5: Sử dụng thẻ ngữ nghĩa (header, nav, section, footer).
- Bootstrap 5: Sử dụng Grid System để chia layout, các component có sẵn (Navbar, Card, Modal, Carousel). Bắt buộc Responsive trên Mobile và Tablet.
- CSS: Tùy biến lại màu sắc, font chữ để không bị "giống hệt" giao diện mặc định của Bootstrap.
- jQuery: Xử lý ít nhất 2 sự kiện tương tác (VD: click, hover, scroll, form validation, filter dữ liệu).
- Số lượng trang: Tối thiểu 3 trang (ví dụ: Trang chủ, Danh sách, Chi tiết).
